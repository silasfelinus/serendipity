#Roadmap#

Core Features:
Modern board game feel with a friendly interface
Livechat
Wonderwidgets - Drag-and-drop interface for chained api calls
User registration and login
Stylishly formatted database CRUD access
Backend and frontend API chaining system
Secure and convenient user input for OpenAI keys

Core WonderWidgets:
Wonderwidget - Our core function, a smartscreen that can act as an openai chat window, a text interface, a livechat, a smart gallery, or one chain in a web of api calls
WonderWidgets Lab: Our friendly-gui featuring a 3x3 grid for building, editing, and sharing WonderWidgets. (and yes, the lab itself is, of course, a Wonderwidget)
ChatWidgets: Wonderwidget as a Multi-room livechat interface with openai integration
WidgetBots: custom widgets with friendly avatars that connect to external api calls, (usually Chatgpt instance, text or image model processor or hugging face repo)
artWidgets: Custom art libraries generated with WonderWidgets.
Textwidgets: Our tiniest wonderWidget, these are loveably text datasets that function as wildcard libraries to be used inside WonderWidget text prompts
Widget Wonderporium: A web gallery and marketplace of publicly shared WonderWidgets.
WonderPrompts: Our playful rerigerator magnet inspired gui interface for saving and sharing textWidgets

Extended Core:
PromptWidgets: a gui-based Smart text processor for making Wonderwidgets
Serendipity, The Adventure-Game-Slash-Task-Manager: Our flagship superWidget (a combined set of Wonderwidgets)
